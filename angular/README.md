# HYD ERP System

A comprehensive Enterprise Resource Planning (ERP) solution built on modern web technologies.

## 🏗️ Technology Stack

- **Framework**: ABP Framework (LGPL License)
- **Frontend**: Angular 20+ with TypeScript
- **Theme**: Lepton X by Volosoft
- **UI Components**: PrimeNG, Bootstrap 5
- **Styling**: SCSS, Tailwind CSS

## 📦 ERP Modules

### 🏢 HRMS (Human Resource Management)
- Employee Management
- Attendance Tracking
- Leave Management
- Payroll Processing

### 💰 Financial Management
- Accounting & Bookkeeping
- Invoice & Payment Management
- Expense Tracking
- Financial Reports

### 📦 Inventory & Logistics
- Stock Management
- Purchase Orders

- Delivery Tracking

### 🛒 Sales & CRM
- Customer Management
- Lead Tracking
- Opportunity Management
- Sales Pipeline

### 🏭 Manufacturing

### 📊 Product Management
- Product Catalog

- Category Management
- Stock Tracking

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ 
- Angular CLI
- .NET 8+ (for backend)

### Installation
```bash
# Install dependencies
npm install

# Start development server
ng serve

# Build for production
ng build --configuration production
```
